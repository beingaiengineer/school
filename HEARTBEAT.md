# Agent Execution Checklist (HEARTBEAT.md)

This file contains the **execution checklist** that autonomous agents must follow every time they wake up for a session.

## Every execution:

1. **Read `AGENTS.md`** — Understand the repository rules and operating constraints.
2. **Read assigned task** — Check your memory/queue for the next batch of tasks.
3. **Read `CONTENT.md`** — If generating or modifying course content, follow the lesson standards.
4. **Read `DESIGN.md`** — If modifying UI, layout, or MDX components, follow the design system.
5. **Read `QUALITY.md`** — Before finishing work, ensure you pass the validation gates.
6. **Run build/typecheck** — Ensure the repository compiles cleanly.
7. **Update memory** — Mark tasks as done and update your session state.
8. **Commit & PR** — Commit using repository conventions and raise a PR.
9. **Exit** — End the session cleanly.
