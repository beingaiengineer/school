# CI/CD Pipelines Course Design & Outline Spec

This document defines the curriculum map for the CI/CD Pipelines course on BAE School.

## Folder Structure

The course must be organized under `docs/school/cicd-pipelines/`.

## Topic List

- what-is-cicd.mdx
- github-actions-fundamentals.mdx
- building-your-first-pipeline.mdx
- automated-testing-in-ci.mdx
- code-quality-gates.mdx
- artifact-management.mdx
- dockerfile-fundamentals.mdx
- docker-compose-in-ci.mdx
- pipeline-security.mdx
- pipeline-optimization.mdx

## Content Requirements

Every topic page must follow the global BAE School 15-section template in `CLAUDE.md`, the platform voice in `soul.md`, and the course-specific rules in `.claude/agents/cicd-pipelines-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
