# Requirements & Estimation Course Design & Outline Spec

This document defines the curriculum map for the Requirements & Estimation course on BAE School.

## Folder Structure

The course must be organized under `docs/school/requirements-estimation/`.

## Topic List

- requirement-gathering.mdx
- product-requirement-documents.mdx
- writing-user-stories.mdx
- acceptance-criteria.mdx
- estimation-techniques.mdx
- planning-poker.mdx
- velocity-and-burndown.mdx
- technical-debt.mdx

- architecture-decision-records.mdx

## Content Requirements

Every topic page must follow the global BAE School 16-section template in `CONTENT.md`, the platform voice in `SOUL.md`, and the course-specific rules in `.claude/agents/requirements-estimation-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
