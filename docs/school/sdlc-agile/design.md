# SDLC & Agile Course Design & Outline Spec

This document defines the curriculum map for the SDLC & Agile course on BAE School.

## Folder Structure

The course must be organized under `docs/school/sdlc-agile/`.

## Topic List

- what-is-sdlc.mdx
- waterfall-vs-agile.mdx
- agile-principles.mdx
- scrum-framework.mdx
- kanban-method.mdx
- sprint-planning.mdx
- daily-standups.mdx
- sprint-retrospectives.mdx
- sprint-reviews-and-demos.mdx
- definition-of-done.mdx

## Content Requirements

Every topic page must follow the global BAE School 15-section template in `CLAUDE.md`, the platform voice in `soul.md`, and the course-specific rules in `.claude/agents/sdlc-agile-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
