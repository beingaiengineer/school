## 2. Engineering Story

A data science team at a mid-size analytics firm wanted to add an AI assistant that could answer questions about their internal datasets — not just look things up, but actually run code. LangChain felt heavyweight for their use case, and OpenAI function calling required a lot of scaffolding to make code execution safe.

They discovered HuggingFace's SmolAgents. Its core idea was radical simplicity: the agent's "tool" is a Python code interpreter. Instead of calling a function, the LLM writes a small Python snippet, SmolAgents executes it in a sandboxed environment, captures the output, and feeds it back. The team connected it to their pandas DataFrames and within a day had an assistant that could answer "What was our highest revenue quarter?" by actually writing and running `df.groupby('quarter')['revenue'].sum().idxmax()`. No complex tool schemas. No JSON parsing hell. Just the LLM writing code, and SmolAgents running it.
