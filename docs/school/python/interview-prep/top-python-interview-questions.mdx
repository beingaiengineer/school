## 6. Beginner Example

If a question asks whether Python variables hold values or references, answer from behavior:

```python
a = []
b = a
b.append(1)
print(a)
```

Output:

```text
[1]
```

That tells you both names point to the same list object.

## 11. Tricky Questions & Edge Cases

| Trap | Why It Matters |
|---|---|
| Same operator, different types | `+` adds numbers but joins strings |
| Wrong `pip` | Package installs into another Python |
| Import-time work | Can break startup and tests |
| Mutable references | One change can appear through many names |
