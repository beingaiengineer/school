## 2. Engineering Story

A seasoned Java developer transitioned to Python and deployed a microservice with a function that used a mutable list as a default argument `def add_item(item, cache=[])`. After running perfectly in staging, production traffic hit, and users suddenly started seeing each other's private data mixed together. A grueling debugging session revealed the classic Python trap: default arguments are evaluated only once at definition time, creating a shared state across all function calls.
