## 6. Beginner Example

```python
arr = [[]] * 3
arr[0].append(1)
print(arr)
```

Output: `[[1], [1], [1]]`

## 15. Common Mistakes

- Using mutable defaults.
- Using `is` instead of `==`.
- Reusing exhausted generators.
- Confusing shallow and deep copy.
