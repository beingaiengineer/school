# OOP & Design Patterns Course Design & Outline Spec

This document defines the curriculum map for the OOP & Design Patterns course on BAE School.

## Folder Structure

The course must be organized under `docs/school/oop-patterns/`.

## Topic List

- oop-in-production.mdx
- classes-and-objects-real-world.mdx
- inheritance-vs-composition.mdx
- solid-principles.mdx
- design-patterns-strategy.mdx
- repository-pattern.mdx
- service-layer-pattern.mdx
- dependency-injection.mdx
- clean-architecture.mdx

## Content Requirements

Every topic page must follow the global BAE School 16-section template in `CONTENT.md`, the platform voice in `SOUL.md`, and the course-specific rules in `.claude/agents/oop-patterns-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
