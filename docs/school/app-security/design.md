# App Security Course Design & Outline Spec

This document defines the curriculum map for the App Security course on BAE School.

## Folder Structure

The course must be organized under `docs/school/app-security/`.

## Topic List

- security-mindset.mdx
- threat-modeling.mdx
- owasp-top-10.mdx
- authentication-deep-dive.mdx
- authorization-rbac-abac.mdx
- secrets-management.mdx
- api-security.mdx
- dependency-security.mdx
- security-in-cicd.mdx
- data-protection.mdx

## Content Requirements

Every topic page must follow the global BAE School 15-section template in `CLAUDE.md`, the platform voice in `soul.md`, and the course-specific rules in `.claude/agents/app-security-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
