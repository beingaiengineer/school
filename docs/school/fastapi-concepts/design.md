# API Concepts & FastAPI Course Design & Outline Spec

This document defines the curriculum map for the API Concepts & FastAPI course on BAE School.

## Folder Structure

The course must be organized under `docs/school/fastapi-concepts/`.

## Topic List

- what-is-an-api.mdx
- rest-api-design.mdx
- fastapi-fundamentals.mdx
- request-response-models.mdx
- pydantic-validation.mdx
- crud-operations.mdx
- authentication-jwt.mdx
- middleware-and-cors.mdx
- error-handling-in-apis.mdx
- background-tasks.mdx
- websockets.mdx
- rate-limiting.mdx

## Content Requirements

Every topic page must follow the global BAE School 15-section template in `CLAUDE.md`, the platform voice in `soul.md`, and the course-specific rules in `.claude/agents/fastapi-concepts-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
