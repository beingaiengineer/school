# Git & Workflows Course Design & Outline Spec

This document defines the curriculum map for the Git & Workflows course on BAE School.

## Folder Structure

The course must be organized under `docs/school/git-workflows/`.

## Topic List

- git-fundamentals.mdx
- branching-strategies.mdx
- pull-requests.mdx
- code-review-culture.mdx
- merge-conflicts.mdx
- git-rebase-vs-merge.mdx
- git-hooks-and-automation.mdx
- monorepo-vs-polyrepo.mdx
- git-advanced-operations.mdx
- conventional-commits.mdx

## Content Requirements

Every topic page must follow the global BAE School 15-section template in `CLAUDE.md`, the platform voice in `soul.md`, and the course-specific rules in `.claude/agents/git-workflows-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
