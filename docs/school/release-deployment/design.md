# Release & Deployment Course Design & Outline Spec

This document defines the curriculum map for the Release & Deployment course on BAE School.

## Folder Structure

The course must be organized under `docs/school/release-deployment/`.

## Topic List

- deployment-fundamentals.mdx
- environments-dev-staging-prod.mdx
- deployment-strategies.mdx
- blue-green-deployment.mdx
- canary-deployment.mdx
- feature-flags.mdx
- release-management.mdx
- rollback-strategies.mdx
- infrastructure-as-code.mdx
- terraform-basics.mdx

## Content Requirements

Every topic page must follow the global BAE School 15-section template in `CLAUDE.md`, the platform voice in `soul.md`, and the course-specific rules in `.claude/agents/release-deployment-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
