Every topic page must follow the global BAE School 15-section template in `CLAUDE.md`, the platform voice in `soul.md`, and the course-specific rules in `.claude/agents/sli-slo-agent.md`.

Each page must also include the course-specific `## Top GitHub Repos` section required by the agent instructions.
