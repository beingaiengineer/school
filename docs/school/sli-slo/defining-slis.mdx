## 2. Engineering Story

When Netflix began scaling its global streaming architecture, they realized that traditional server metrics like CPU utilization or database latency were terrible indicators of user happiness. A server could be at 20% CPU, but users might still be seeing a buffering wheel. To solve this, Netflix defined a core, user-centric Service Level Indicator (SLI): 'Stream Starts Per Second' (SPS). SPS measures the exact rate at which users successfully press play and a video begins. If SPS drops globally, Netflix knows users are failing to watch videos, regardless of what the backend infrastructure metrics say. This shift to measuring the actual user journey became the gold standard for defining meaningful SLIs.
