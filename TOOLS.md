# BAE School Tool Knowledge (TOOLS.md)

This file lists the foundational tools and technologies used within the BAE School repository. Agents should consult this list when executing tasks.

## Available Tools

- **Docusaurus**: The static site generator used for the documentation platform.
- **MDX**: Markdown with JSX components, used for all course content pages.
- **LearningFlow**: Custom React component (`<LearningFlow>`) used for all technical concept diagrams.
- **React**: The underlying UI framework.
- **TypeScript**: Used for all custom components and configuration files.
- **npm**: The package manager (use `npm run build`, `npm run typecheck`).
- **GitHub**: Used for CI/CD and pull request workflows.
- **Illustration Pipeline**: Custom agentic workflow (via `bae-illustrations` skill or headless queue) to generate character art.
